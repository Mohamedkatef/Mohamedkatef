### Highlights @ Boston Scientific

- 🧠 **FaraStar PFA — Real-Time Bio-Impedance Sensing with ML.** Co-developed the contact- and lesion-sensing platform on the FaraStar Pulsed Field Ablation system. Three-class catheter-contact classification (no contact / bad contact / good contact) and binary lesion-state classification (reversible / irreversible electroporation), trending at **90%+ accuracy** and delivering live feedback during PFA procedures for the first time. Designed the phase-referenced differential-voltage signal-acquisition system from scratch; co-built the end-to-end ML pipeline in Python (data prep → feature engineering → neural network → in-procedure inference).
- 🧪 **Flow Model — Novel Benchtop Simulation for Cardiac EP.** Conceived, designed, and built a clinical/benchtop simulation model replicating septal puncture and transcatheter EP access routes. **Eliminated ~$892K in preclinical animal-study costs**, accelerated DV iteration cycles, and earned the company-wide **Clinical Innovation Prize**.
- 🫧 **Intracardiac Bubble-Formation Study.** Designed and built an airtight benchtop model integrating bubble sensors to quantify intracardiac air-embolism formation during transseptal puncture. Benchmarked VersaHub & VersaCross against competitor systems (Atraverse, Hotwire) and characterized AutoStop's effect on bubble formation — direct patient-safety implications. Owned the full study design end-to-end.
- ⚡ **AutoStop & Continuous Impedance Monitoring (CIM).** Contributed to two first-in-class VersaHub features from clinical need through validated implementation. Built custom DV test fixtures (AutoStop Box, CIM Box, Arduino-controlled RS dongle) and verified sensing/behavior thresholds for impedance, current, and phase.
- 📐 **VersaHub Usability Engineering Plan (IEC 62366-1).** Authored the formal Usability Engineering Plan that governs all physician-facing UI design decisions for RFP-NXT — formative/summative methodology, use-error mitigation, UX traceability for novel features.
- ✅ **RFP-100A LMMB Design Validation Campaign.** Led end-to-end: test scope, resource allocation, tester assignments, execution.
- 📑 **510(k) Submission Support & Risk Management.** Systems-engineering documentation, DHF traceability, design FMEA and ISO 14971 risk activities across VersaHub and FaraStar.
